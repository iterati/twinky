- repeats - [0, 1] -> [0, 1], [0, 1]
- mirrored - [0, 1] -> [0, 1], [1, 0]
*** Height dependent
- spiral
- distored

- spin

- door - same as window really
- ncolors - more than one window with set ratio
- panels - same as ncolors really
- falling - color falls from sky!
